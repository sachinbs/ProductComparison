Project_Sentiment_Analysis:
---------------------------
This code is used to collect the twitter data for search queries: #Samsung and #Apple.
To run this code successfully, follow the steps below:
1. Create a Twitter Developer account.
2. On line 17 and 18, enter your account's 'Consumer Key' and 'Consumer Secret'

--------------------------------------------------------------------------------------------------------------------------------

Function 'collecttwittertweets()' will generate a 'collected_twitter_tweets.txt' file in the current folder.

The data used for our demo is contained in 'our_collected_twitter_tweets.txt' file, and this data is used henceforth in the 
code.

After the NLTK is successfully run, the final dataframe is stored in 'tokenizedDataframe.pickle' file which is used for the 
analysis.
